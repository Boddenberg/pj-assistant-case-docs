# O início do PJ Assistant

Este é o primeiro post do blog de desenvolvimento do **PJ Assistant** — um assistente bancário inteligente para Pessoas Jurídicas.

<!-- truncate -->

## O Desafio

Construir um sistema completo que integra:

- **IA Conversacional** com RAG para contextualização
- **Operações bancárias** reais (Pix, extratos, boletos)
- **Interface mobile-first** moderna e responsiva

## Stack Escolhida

A escolha de cada tecnologia foi deliberada:

| Tecnologia | Motivo |
|-----------|--------|
| **Go** | Performance, concorrência nativa, tipagem forte |
| **Python** | Ecossistema ML/AI maduro (LangChain, ChromaDB) |
| **React Native** | Cross-platform, TypeScript, ecossistema rico |
